A resource index for concrete removal and disposal scopes before replacement or repair.

## Resource Sequence

- Start with dimensions, project type, and site conditions before estimating.
- Separate concrete quantity, base prep, delivery, labor, and special details.
- Use related references to compare assumptions before ordering material or requesting bids.
- Confirm final requirements with local suppliers, contractors, inspectors, or engineers when needed.
